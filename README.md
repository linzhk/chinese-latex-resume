# chinese-latex-resume
全中文汉化latex简历。支持overleaf个性化编辑并生成pdf。适用于互联网求职产品、运营、算法、开发岗
## 效果图（截图原因，实际上没那么糊）

## overleaf使用方式
1、 将.tex文件上传到overleaf，并且在指定位置改成自己的部分，或者复制相应的代码块以增加经历
2、 使用xlatex运行（在左上角菜单选择xlatex运行，而不是用pdflatex）
3、 下载你的pdf
